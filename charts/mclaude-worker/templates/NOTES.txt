=== Next steps ===

1. Read the NKey public key generated by the pre-install Job:

   kubectl logs job/{{ .Release.Name }}-gen-host-nkey -n {{ include "mclaude-worker.namespace" . }}

2. Register the cluster host with the control-plane from your workstation:

   mclaude host register --type cluster --name {{ .Values.host.name | quote }} --nkey-public <pubkey>

   Replace <pubkey> with the public key printed above (starts with "U").

The controller will not connect to hub NATS until the host is registered and the
control-plane has issued a host JWT. Until then the controller pod will log:

  NKey <pubkey> not registered with control-plane. To complete registration run:
    mclaude host register --type cluster --name <display-name> --nkey-public <pubkey>

and retry every 5–60 seconds (exponential backoff).
